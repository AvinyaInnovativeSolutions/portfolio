CHEMeaseTry

CHEMeaseTry is an application built using python which can display certain details of an element upon the entry of its atomic number.
The first step was to create a class of an atom and define its properties and functions.
The first function would return the electronic configuration of the element in the form of a list.
Using the data obtained from the previous function, we can now draw the schematic diagram of the element using the turtle library in python.
I have also sourced a csv file from www.kaggle.com which contains certain details which are unique to certain elements and cannot be obtained by mere mathematical calculations.
From the csv file wfe can obtain the element name, atomic mass etc. The element name will be displayed on the title bar of the turtle window.
